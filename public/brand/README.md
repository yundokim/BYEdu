# 브랜드 자산

이 폴더에 로고 이미지를 두면 사이트 헤더·푸터에 자동 적용됩니다.

## 로고 적용 방법

1. SVG 또는 PNG 파일을 이 폴더에 복사하세요.
   예) `public/brand/logo.svg`

2. `content/site.ts` 를 열어 `brand.logo.src` 에 경로를 입력하세요.
   ```ts
   brand: {
     logo: {
       src: "/brand/logo.svg", // ← public/ 이후 경로
       alt: "백양에듀 로고",
     },
   }
   ```

3. 저장하면 dev 서버에서 즉시 반영됩니다.

## 권장 사양

- **형식**: SVG 권장 (PNG 도 무관 — 가능하면 투명 배경)
- **비율**: 1:1 정사각형이 가장 깔끔합니다 (헤더에서 36×36 원형 마크로 노출)
- **여백**: 로고 자체에 약간의 inner padding 을 두면 원형 컨테이너 안에서 잘 보입니다.

## 로고를 빼고 싶다면

`brand.logo.src` 를 빈 문자열 (`""`) 로 두면 자동으로 한자(白) 폴백 마크가 노출됩니다.
