# 컨설턴트 사진

이 폴더에 컨설턴트 사진을 두면 컨설턴트 카드에 자동 적용됩니다.

## 사진 적용 방법

1. JPG / PNG / WebP 파일을 이 폴더에 복사하세요.
   예) `public/consultants/lee-jihoon.jpg`

2. `content/site.ts` 의 `consultants.list` 에서 해당 컨설턴트의 `photo` 필드에 경로를 넣으세요.
   ```ts
   consultants: {
     list: [
       {
         name: "이지훈",
         photo: "/consultants/lee-jihoon.jpg", // ← public/ 이후 경로
         // ...
       },
     ],
   }
   ```

3. 저장하면 dev 서버에서 즉시 반영됩니다.

## 권장 사양

- **비율**: 9:16 세로형 (예: 900×1600 또는 1080×1920) — 카드가 세로 포트레이트로 노출됩니다
- **포맷**: JPG 또는 WebP (얼굴이 또렷한 인물 사진)
- **분위기**: 차분한 배경 / 무릎~허리부터 머리까지 / 자연광 권장
- **얼굴 위치**: 상단 1/3 지점 (배지가 좌상단에 겹치므로 얼굴은 중앙~약간 우측 권장)
- **용량**: 페이지 성능을 위해 사진 1장당 500KB 이하 권장 (필요 시 [Squoosh](https://squoosh.app) 등으로 압축)

## 사진을 비워두면

`photo: ""` 로 두면 한자(姓) + 그라디언트 카드가 자동 폴백으로 노출됩니다.
사진 준비 전이라도 디자인이 깨지지 않습니다.

## 컨설턴트 추가/제거

`content/site.ts` 의 `consultants.list` 배열에 객체를 추가/삭제하세요.
3명 이상이 되면 모바일은 1열, 데스크탑은 자동으로 그리드가 조정됩니다.
