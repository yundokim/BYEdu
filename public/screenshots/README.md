# 카카오톡 대화 캡처

특장점(Why) 섹션 우측에 노출되는 실제 카카오톡 대화 캡처 이미지를 두는 폴더입니다.

## 적용 방법

1. 캡처 파일 두 장을 이 폴더에 복사합니다.
   ```bash
   cp ~/Downloads/parent-chat.png  public/screenshots/chat-1.jpg
   cp ~/Downloads/student-chat.png public/screenshots/chat-2.jpg
   ```

2. [`content/site.ts`](../../content/site.ts) 의 `why.screenshots` 배열에서 각 항목의 `src` 에 경로 입력:
   ```ts
   screenshots: [
     {
       src: "/screenshots/chat-1.jpg",
       alt: "학부모님과의 실제 카카오톡 대화",
       caption: "학부모님과의 실시간 소통",
     },
     {
       src: "/screenshots/chat-2.jpg",
       alt: "학생과의 실제 카카오톡 대화",
       caption: "학생의 질문에 즉각 응답",
     },
   ],
   ```

3. 저장 → dev 서버에서 즉시 반영됩니다.

## ⚠️ 개인정보 보호 — 업로드 전 필수 작업

실제 대화 캡처에는 개인정보가 포함될 수 있습니다. 업로드 전에 반드시 다음을 가리세요.

- 학생/학부모 **실명** → 익명 처리 또는 ‘ㅇ어머님’, ‘ㅇ학생’ 등으로 대체
- **프로필 사진 / 프로필명** → 모자이크 또는 기본 이미지로 교체
- **연락처 / 전화번호 / 학교명** → 모자이크
- **민감한 학습 정보** (구체적 성적 등) → 학생·학부모 사전 동의 받은 내용만 노출

[Squoosh](https://squoosh.app) 또는 모자이크 가능한 모바일 앱(예: "사진 모자이크")으로 손쉽게 처리할 수 있습니다.

## 권장 사양

- **비율**: 3:4 세로형 (예: 900×1200) — 카드에 정확히 들어맞습니다.
- **포맷**: JPG / PNG / WebP. 텍스트 가독성이 중요하므로 화질은 80% 이상 권장.
- **용량**: 1장당 400KB 이하 권장 (텍스트가 또렷이 보이는 한도 내에서 압축).
- **크롭**: 가장 임팩트 있는 대화 구간(평일 늦은 시간 / 주말 / 마감 직전 응답 등)을 중심으로 3:4 비율로 자르세요.

## 이미지를 비워두면

`src: ""` 로 두면 같은 위치에 placeholder 카드(아이콘 + 사용 안내) 가 노출됩니다. 사진을 준비하기 전이라도 디자인이 깨지지 않습니다.

## 사진 개수를 늘리거나 줄이려면

`why.screenshots` 배열에 항목을 추가/삭제하면 됩니다. 그리드는 2열로 고정되어 있으니 짝수(2장 또는 4장)가 가장 깔끔합니다.
