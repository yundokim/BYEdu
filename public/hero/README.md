# 히어로 이미지

메인 페이지 상단 우측에 노출되는 2장의 이미지를 두는 폴더입니다.
데스크탑에서는 좌·우 살짝 stagger 되어 배치되고, 모바일에서는 세로로 쌓입니다.

## 적용 방법

1. 사진 두 장을 이 폴더에 복사합니다.
   ```bash
   cp ~/Downloads/main-1.jpg public/hero/main-1.jpg
   cp ~/Downloads/main-2.jpg public/hero/main-2.jpg
   ```

2. [`content/site.ts`](../../content/site.ts) 의 `hero.images` 배열에서 각 항목의 `src` 에 경로 입력:
   ```ts
   hero: {
     // ...
     images: [
       { src: "/hero/main-1.jpg", alt: "백양에듀 메인 이미지 1" },
       { src: "/hero/main-2.jpg", alt: "백양에듀 메인 이미지 2" },
     ],
   }
   ```

3. 저장 → dev 서버에서 즉시 반영됩니다.

## 권장 사양

- **비율**: 3:4 세로형 (예: 1200×1600) — 카드에 정확히 들어맞습니다.
- **포맷**: JPG / WebP. 인물 사진은 JPG, 그래픽·일러스트는 WebP 권장.
- **분위기**: 컨설팅 장면 / 자료 작업 / 송도 캠퍼스 / 인물 사진 등.
- **두 이미지의 톤 일관성**: 색감·밝기·채도를 비슷하게 맞추면 더 정돈된 느낌.
- **용량**: 1장당 500KB 이하 권장 ([Squoosh](https://squoosh.app) 등으로 압축).

## 이미지를 비워두면

`src: ""` 로 두면 placeholder 카드(아이콘 + 경로 안내)가 노출됩니다.
사진을 준비하기 전이라도 디자인이 깨지지 않습니다.

## 사진 개수 조절

`hero.images` 배열에 항목을 추가/삭제하면 그리드가 자동 적응합니다.
- 1장만 쓰고 싶다면 항목 1개로 줄이면 자동으로 한 장만 노출
- 3~4장 사용도 가능하지만, 우측 컬럼 폭이 좁아 2장이 가장 적합합니다.
